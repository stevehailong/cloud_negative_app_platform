## 📚 使用文档

### 基本操作

1. **查看流水线列表**
   - 访问 http://localhost/pipelines
   - 页面显示所有流水线
   - 可以看到最近执行状态

2. **筛选流水线**
   - 输入名称进行搜索
   - 按类型筛选（构建/CI-CD/测试）
   - 按状态筛选（启用/禁用）

3. **创建流水线**
   - 点击"新建流水线"
   - 填写必填项：名称、编码、关联应用、类型、CI工具
   - 可选填写JSON配置
   - 点击确定保存

4. **执行流水线**
   - 找到要执行的流水线
   - 点击"执行"按钮
   - 确认后触发执行
   - 执行状态会实时更新

5. **查看执行记录**
   - 点击流水线的"记录"按钮
   - 右侧抽屉显示所有执行历史
   - 可以查看详情、日志
   - 可以停止运行中的执行

6. **管理流水线**
   - 编辑：修改流水线配置
   - 启用/禁用：通过开关切换
   - 删除：删除不需要的流水线

## 🔧 技术细节

### 组件结构
```
PipelineList.vue (主页面)
  ├── el-table (列表)
  ├── el-dialog (创建/编辑)
  └── el-drawer (执行记录)
       └── PipelineRuns.vue (子组件)
            ├── el-table (执行记录列表)
            └── el-dialog (执行详情)
```

### 状态管理
- 使用 Vue 3 Composition API
- reactive() 管理查询参数
- ref() 管理组件状态
- 无需Pinia全局状态

### API调用
- Axios封装统一请求
- 自动添加JWT Token
- 统一错误处理
- 自动消息提示
