## 🔍 故障排查

### 如果页面显示"功能开发中，敬请期待"

这说明看到的还是旧版本的占位页面。解决方法：

```bash
# 1. 清除浏览器缓存
# Chrome: Ctrl+Shift+Delete 或 Cmd+Shift+Delete
# 勾选"缓存的图片和文件"
# 点击"清除数据"

# 2. 硬刷新页面
# Windows: Ctrl+Shift+R
# Mac: Cmd+Shift+R

# 3. 如果还不行，重启浏览器
```

### 如果页面空白或报错

```bash
# 1. 打开浏览器开发者工具（F12）
# 2. 查看Console标签是否有错误
# 3. 查看Network标签，检查API请求

# 4. 重新创建容器
docker-compose up -d --force-recreate frontend

# 5. 清除浏览器缓存后重试
```

### 如果显示404

```bash
# 检查路由配置
# 访问主页后点击侧边栏"流水线"菜单
# 或直接访问 http://localhost/pipelines
```

## 📊 API测试

如果页面看不到数据，可以直接测试API：

```bash
# 1. 先登录获取token（在浏览器登录后，从localStorage获取）
# 或使用curl登录
TOKEN=$(curl -s -X POST http://localhost/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}' \
  | jq -r '.data.token')

# 2. 测试流水线API
curl http://localhost/api/v1/pipelines \
  -H "Authorization: Bearer $TOKEN" \
  | jq .

# 预期返回包含5条流水线的数据
```
